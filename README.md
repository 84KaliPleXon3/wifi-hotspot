# WiFi Hotspot for Linux :penguin:
Create a Wi-Fi hotspot easily!

## Installation
```sh
git clone https://github.com/mateus4k/wifi-hotspot
cd wifi-hotspot
sudo chmod 755 wifi-hotspot.sh
```
## Usage example
Enable:
```sh
./wifi-hotspot.sh wlxf8d1111c9141 "John's Hotspot" "password"
```
Disable: 
```sh
./wifi-hotspot.sh -d
```
